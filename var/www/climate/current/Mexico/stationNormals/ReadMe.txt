There are 4 files, one for each data element. Each file has the same format:

Field 1: State code (MX for Mexico, CN for Canada, US for United States)
Field 2: Station Number
Field 3: Longitude in decimal degrees
Field 4: Latitude in decimal degrees
Field 5: Elevation in meters
Fields 6-17: Monthly values

-99.0 is used for a missing value code.

westNA: Western North America Extent (includes Western Canada and Alaska).

MMONTH.dat  minimum temps, degrees C
TMONTH.dat  average temps, degrees C
XMONTH.dat  maximum temps, degrees C
PMONTH.dat  total precipitation, mm.

Mexico: Mexico, the southern tier of the United States, and Cuba

MMONTH.dat  minimum temps, degrees C
TMONTH.dat  average temps, degrees C
XMONTH.dat  maximum temps, degrees C
PMONTH.dat  total precipitation, mm.
