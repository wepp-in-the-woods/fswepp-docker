Importing the ASCII Grid data into ArcView 3.x

  1. save as file on file system accessible to ArcView eg my_deposition.asc
  2. Open ArcView
  3. activate the Spatial Analyst extension
  4. Invoke "Import Data Source" tool
  5. Select "ASCII Raster" import file type
  6. Select the saved ASCII raster file eg my_deposition.asc
  7. Import file as a Theme into the View
========================

ESRI ArcGIS 9.x

  1. save as file on file system accessible to ArcGIS eg my_deposition.asc
  2. Using ArcToolbox,
    1. select Conversion Tools > To Raster > ASCII to Raster
    2. Provide path to ASCII file (eg my_deposition.asc)
    3. Provide path to Output Raster (eg my_deposition)
    4. click OK -- after converting to Raster the output raster (eg my_deposition) should automatically load into ArcMap
  3. Export output Raster file (eg my_deposition) into ArcGIS GRID;
    1. right click on the file to Export
    2. provide output filename (eg my_deposition_a)
    3. select GRID format
  4. Set up the projection
    1. Download file prj.adf from https://daac.ornl.gov/cgi-bin/MODIS/GR_common/prj.adf
    2. copy the prj.adf file into the GRID file directory (my_deposition_a) created in the previous step
  5. Load the GRID file in ArcMap -- Make sure that the projection is defined on the dataset.
  6. If the projection isn't defined, use the following file to define the projection.
    1. Download file modis.prj from https://daac.ornl.gov/cgi-bin/MODIS/GR_common/modis.prj
    2. Copy this file into <ESRI installation Folder>\Coordinate Systems\Projected Coordinate Systems\World\Sphere-based
    3. manually set the projection for the file
