WEPP:Road Batch soil texture

Soil texture parameter values depend on the nominal soil texture. Some parameter values also depend upon
  • road design,
  • road surface,
  • traffic level, and
  • rock fragment content
Any of the above conditions may vary within a given batch run.

In order to see the soil properties for a given condition, the user should go to WEPP:Road, select a set of input conditions, and click Soil Texture to get a table of the soil parameter values used for any combination of soil texture, road surface, and road design.
