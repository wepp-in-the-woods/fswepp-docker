WEPP Resources

FS WEPP

  • FS WEPP interfaces (WEPP:Road, Disturbed WEPP, ERMiT, FuME, Tahoe Basin Model, and others) (USDA FS RMRS Moscow)
  • Cross-Drain Spacing -- Sediment Yield Program [ metric units | English units ]
  • Disturbed WEPP batch and Batch ERMiT interface spreadsheets download (USDA FS RMRS Moscow)
  • GL Interface (WEPP Watershed Online Interface)
  • WEPP Post-Fire Erosion Prediction (PEP) interface -- a computer simulation tool that estimates soil erosion following an actual or simulated wildfire from soil burn severity maps
  • Lake Tahoe WEPP Watershed GIS Interface

WEPP

  • WEPP software and associates (USDA ARS NSERL)
  • WEPP for Windows download (USDA ARS NSERL)
  • WEPP Model Documentation (NSERL Report #10, July 1995, and related) (USDA ARS NSERL)
  • Web interface to WEPP (prototype) -- rangeland and cropland (USDA ARS NSERL)

GeoWEPP

  • Database to support post-wildfire analysis with GeoWEPP
  • GeoWEPP: The Geo-spatial interface for the Water Erosion Prediction Project (SUNY-Buffalo & Federal Agencies)
  • Download GeoWEPPer -- a prototype Perl script to allow you to use GeoWEPP partial results as inputs to WEPP FuMe or Disturbed WEPP (needs GeoWEPP and a Perl interpreter installed)

CLIGEN resources

  • Cligen Weather Generator, expanded and improved (USDA ARS NSERL)
  • Create a Cligen station file from online GDS data (international daily temperature and precipitation data) (USDA ARS NSERL)
  • ARS Water Database (USDA ARS Hydrolab)
    • Historical NWS and GDS climate data for the United States and Worldwide sites (USDA ARS Hydrolab)

PRISM resources

  • PRISM (Spatial Climate Analysis Service) (Oregon State University)
