Batch ERMiT interface spreadsheet

Optional ArcInfo Hillslope Delineation Toolbox (tbx) to gather some parameter values for the batch interfaces

The interface spreadsheets are Microsoft Excel spreadsheets that you download and run on your PC. They are designed to make running multiple Disturbed WEPP or ERMiT scenarios much more efficient.

The optional Toolbox is an ArcInfo .tbx file that one downloads and installs on one's PC. The toolbox is not required for running the Disturbed WEPP Batch or Batch ERMiT interface spreadsheets, but it can assist in collecting the run parameter input values.
