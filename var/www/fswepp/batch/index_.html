These are Microsoft Excel spreadsheets that you download and run on your PC. They are designed to make running multiple Disturbed WEPP or ERMiT scenarios much more efficient.

Requirements
  • Microsoft Excel version 97-2003 or newer
  • For custom ArcInfo Hillslope Delineation Toolbox (optional):
    • ESRI ArcGIS 9.x and ArcGIS Spatial Analyst extension
    • GIS datasets in one common coordinate system
      • an elevation raster, such as a 10-meter DEM or LiDAR, as one single raster grid (publicly available)
      • polygons describing stream centerline data with names (publicly available)
      • polygons describing treatment types (user-developed)
      • polygons describing land use (user-developed)
  • A connection to the Internet, with a (relatively) static IP address, as the modeling is done on Forest Service servers.
