WEPP on the Web

If your browser has JavaScript enabled, you should have been shown a JavaScript "alert box" upon pressing the "Test for JavaScript" link on the previous page.

If you did not get that alert box, it appears that your browser is not taking JavaScript suggestions.

Here's what you can do...

  • Netscape Navigator
    • Version 4 (Communicator)
      1. From the Edit menu, select Preferences....
      2. Click the Advanced category on the left side of the screen.
      3. Check the Enable Javascript box.
      4. Click OK.
    • Version 3
      1. From the Options menu, select Network preferences.
      2. Click the Languages tab.
      3. Check the Enable Javascript box.
      4. Click OK.
  • Microsoft Internet Explorer
    • Version 5
      1. From the Tools menu, select Internet Options....
      2. Click the Security tab.
      3. Select the appropriate "Web content zone":
        • Click on Internet if you are running FS WEPP over the Internet
        • Click on Local intranet if you are running FS WEPP locally
      4. Click on the Custom Level... button
      5. Enable Run ActiveX controls and plugins
      6. Enable Script ActiveX controls marked safe for scripting
      7. Click OK.
      8. Click OK.
    • Version 3
      1. From the View menu, select Options....
      2. Click the Security tab.
      3. In the Active Content section, check the Run ActiveX scripts box.
      4. Click OK.
