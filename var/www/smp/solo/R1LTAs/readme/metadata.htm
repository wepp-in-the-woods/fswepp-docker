  Description
    Abstract:
        Landtype association mapping was done at a scale of
        1:100,000 by Forest Soil Scientists using a Regionally
        consistent legend for dominant groups of landforms 
        and geologic materials occurring in repeatable patterns
        on National Forest Land. 
    Purpose:
        This mapping is intended to support broad-scale watershed
        and landscape level assessments. It should be used as a
        stratification tool for defining locations with similar
        geomorphic environments. Useful for evaluating and comparing
        biophysical conditions across large landscapes. Directly
        linked to distinct stream and riparian types (as controlled)
        by geomorphic properties. Also useful for comparisions of
        erodability, sediment production and delivery.
